前期准备

只是个前期准备就把我给弄的晕头转向的 好了 写个记录已备忘记

服务器准备要求 apache  php  mysql

在腾讯云的服务器刚开始安装的是代环境的 但谁让自己晕呢？ 就是找不到www文件夹 以及mysql密码

后来试了很多次后 抱着学习的心态 就心一横 安装原版centos 自己来配置环境

可是啊。。。。。。    自己在尝试安装了一会后突然发现它的镜像是自带mysql的 吗德 你知道我在现在的心情吗？。。。。

自带mysql  那密码是什么呢？一般root 先试一试吧  可是它出现了这个

ERROR 1045 (28000): Access denied for user ‘root’@’localhost’ (using password: NO)

试了很多遍 我一直以为是自己的密码不对 尝试了很多解决办法

后来就谷歌吧  最终找到了解决办法

什么原因我也就先不去纠结了 毕竟最近还要复习考试。

# /etc/init.d/mysqld stop

# mysqld_safe –user=mysql –skip-grant-tables –skip-networking &

然后打开另一个ssh 登陆

# mysql

mysql>use mysql；

mysql>update user set password=password(“123456″) where user=”root”;

mysql>flush privileges;

mysql>exit；

好了 现在mysql可以登陆了
开始

下载合适的cms（如海洋cms）根据指导把文件放到www根目录下

然后打开自己的域名按照提示安装 记得修改一下自己刚放到www下的相对文件权限

好了 一个视频网站就建好了  但是。。。。
