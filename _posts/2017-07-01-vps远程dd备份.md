自己的rectified  有台机器之前搞了个vagex挂机 不过挣不到钱而且还挺麻烦的  最近有点“项目”就想把机器腾出来  但rectified没有镜像备份而且之前装挂机的一整套的时候还挺麻烦的  以后还要用的话就还要再装  挺麻烦的 于是就像自己弄个备份

看了半天比较适合我这种懒人整机备份的方法有  clonezilla 和远程dd    我选择了后者

安装备份环境  CentOS 6.5 32bit

## 安装sshfs

sshfs 是一个远程挂载的软件  由于要备份整机由于内存的原因所以不能直接备份到本机  就需要sshfs来挂载一个远程硬盘了

```
yum -y install epel-release    # 安装epel 源码
yum -y install fuse-sshfs       # 安装sshfs 
sshfs root@x.x.x.x（ip）:/home/tecmint/ /mnt/tecmint    将会在本地的 /mnt/tecmint 目录下挂载一个叫远程的一个 /home/tecmint 目录。（不要忘了使用你的 IP 地址和挂载点替换 x.x.x.x）。
umount /mnt/tecmint   #卸载命令/mnt/ttecmint 为本地文件地址
```

好了 df-h  看到本地已经挂载上了一个远程磁盘了

## dd开始备份

dd 的命令也没有了解很多

备份的话就是直接

```
dd if=/dev/需要备份磁盘  of=/root/Downloas/1.img
或者  
dd if=/dev/sr0 of=/root/Downloas/2.img bs=2048 conv=noerror,sync
```
